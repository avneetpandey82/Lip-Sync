# ⚠️ IMPORTANT: Download Rhubarb Binary

This directory needs the Rhubarb Lip Sync binary to function.

## Quick Download (Windows)

1. Visit: https://github.com/DanielSWolf/rhubarb-lip-sync/releases/latest
2. Download: `Rhubarb-Lip-Sync-X.X.X-Windows.zip`
3. Extract and find `rhubarb.exe`
4. Place it HERE as: `rhubarb.exe`

Your final path should be:

```
lip-sync/lib/rhubarb/rhubarb.exe
```

## For Linux/Mac

Download the appropriate binary and place it here as `rhubarb` (no .exe extension).
Make it executable: `chmod +x rhubarb`

## Full Instructions

See the project root file: `RHUBARB_SETUP.md`

---

Without this binary, the phoneme extraction will fail!
